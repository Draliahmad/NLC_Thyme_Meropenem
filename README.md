
## Methods
- Data analysis performed in **R (v4.5.1)** using tidyverse-based workflows.
- Statistical comparisons conducted using:
  - Independent two-sample t-tests
  - Effect size estimation (Cohen’s d)

## Key Outputs
- `outputs/summary_stats.csv` – Descriptive statistics
- `outputs/final_statistical_results.csv` – Final statistical results (means, p-values, effect sizes)
